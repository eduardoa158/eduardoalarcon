# Digital Vision Latam — Vista previa local

## Requisitos
- Python 3 instalado.

## Cómo ver la vista previa
1. Inicia un servidor estático desde la raíz del repositorio:

   ```bash
   python -m http.server 8000
   ```

2. Abre tu navegador en:

   ```
   http://localhost:8000/index.html
   ```

## Navegación en inglés
- Página principal en inglés:

  ```
  http://localhost:8000/en/index.html
  ```
