# Digital Vision Latam (sitio estático)

Sitio 100% estático listo para copiar a `public_html` en Hostinger.

## Estructura recomendada de publicación
Puede publicarlo de dos maneras (ambas incluidas):

1. **Estructura por carpetas** (ejemplo: `/servicios/index.html`).
2. **Archivos directos en raíz** (ejemplo: `/servicios.html`).

Para máxima compatibilidad en hosting compartido, este repositorio ya incluye ambas rutas.

## WhatsApp
Todos los botones `.js-wa` se convierten automáticamente mediante `buildWhatsAppLink(interest, sourcePage)` leyendo UTMs.
