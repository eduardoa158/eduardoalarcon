# Digital Vision Latam (sitio estático)

Sitio 100% estático listo para copiar a `public_html` en Hostinger.

## Estructura
- `index.html` en raíz
- Páginas por carpeta con su `index.html`
- Recursos globales en `/assets` y datos en `/data`

## WhatsApp
Todos los botones `.js-wa` se convierten automáticamente mediante `buildWhatsAppLink(interest, sourcePage)` leyendo UTMs.
